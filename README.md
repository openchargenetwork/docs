# Opencharge Documentation

Documentation for the Opencharge protocol - enabling decentralized payment interoperability and verifiable settlement.

**Docs:** https://docs.opencharge.network
**Website:** https://opencharge.network

## Local Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint):

```
npm i -g mint
```

Run locally:

```
mint dev
```

View at `http://localhost:3000`

## Contributing

1. Fork this repository
2. Create a branch for your changes
3. Make your edits to the MDX files
4. Test locally with `mint dev`
5. Submit a pull request

All pages require frontmatter with `title` and `description` fields.

## Structure

- `api-reference/` - OpenAPI spec and endpoint documentation
- `guides/` - Protocol guides and implementation details
- `docs.json` - Navigation and site configuration
